##  Features

-  Fully responsive — works on mobile, tablet, and desktop
-  Homepage with school overview and highlights
-  About page — school history, vision, and mission
-  News & announcements section
-  Contact page with location and communication details
-  Clean and professional UI suited for an educational institution

##  Getting Started

To run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/justus-ndwigah/kavingoni-sec-website.git

# 2. Navigate into the project folder
cd kavingoni-sec-website

# 3. Open index.html in your browser
open index.html
```

No installations or dependencies required — it's pure HTML/CSS/JS.
